Welcome

This is an agressively styled bootstrap theme with the intention of being a drop in overlay.
All components and utilities are pure CSS, except the custom glass reflections addon documented at the end of this page.
This theme is designed for dark UIs and is fully responsive.

Powered by pure spite Brandon Moore 2025

Features

  • Glassmorphism cards with animated neon accents.
  • Gradient buttons with glossy borders and hover sheen.
  • Chips, badges, and progress bars with glow effects.
  • Utility classes for backgrounds, glows, and text colors.
  • Advanced neon text effects and lens distortion demo.
  • Responsive by default; optimized for dark UI.

Installation

Basic Setup

Add the Neon Glass stylesheet to your existing Bootstrap project. Include it after Bootstrap's CSS to override default styles.

CDN Links
<!-- Bootstrap 5 -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- Neon Glass Theme -->
<link href="css/bootstrap-neon.css" rel="stylesheet">
Optional Dependencies

For advanced features like lens effects and icons:

<!-- Font Awesome Icons -->
<link href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.2/css/all.min.css" rel="stylesheet">

<!-- html2canvas-pro (for lens distortion effects) -->
<script src="https://cdn.jsdelivr.net/npm/html2canvas-pro@1.5.11/dist/html2canvas-pro.min.js"></script>
<script src="js/lens.js"></script>
JavaScript
<!-- Bootstrap Bundle (includes Popper) -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
HTML Template

Complete starter template with all required dependencies and dark theme configuration.

<!DOCTYPE html>
<html data-bs-theme="dark" lang="en">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Neon Glass App</title>

  <!-- Bootstrap CSS -->
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

  <!-- Font Awesome Icons -->
  <link href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.2/css/all.min.css" rel="stylesheet">

  <!-- Neon Glass Theme -->
  <link href="css/bootstrap-neon.css" rel="stylesheet">
</head>
<body>

  <!-- Your content here -->
  <div class="container my-5">
    <div class="card">
      <div class="neon-accent-line"></div>
      <div class="card-header">
        <h5 class="card-title mb-0">
          <i class="fas fa-bolt me-2"></i>
          Welcome to Neon Glass
        </h5>
      </div>
      <div class="card-body">
        <p>Your cyberpunk-themed application starts here.</p>
        <button class="btn btn-primary">Get Started</button>
      </div>
    </div>
  </div>

  <!-- Bootstrap Bundle JS -->
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

  <!-- Optional: Lens Effects -->
  <script src="https://cdn.jsdelivr.net/npm/html2canvas-pro@1.5.11/dist/html2canvas-pro.min.js"></script>
  <script src="js/lens.js"></script>

</body>
</html>
Pro Tip: Set data-bs-theme="dark" on the <html> tag to enable Bootstrap's dark mode and ensure optimal appearance with the Neon Glass theme.
